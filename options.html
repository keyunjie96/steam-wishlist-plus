Steam Cross-Platform Wishlist

Extension settings and data source configuration

Data Source: Wikidata Active

Works out of the box!

This extension uses Wikidata to look up platform availability. No configuration required.

Enhanced Mode: IGDB Optional

For games not found in Wikidata, IGDB can provide additional coverage. This requires a free Twitch Developer application. Create one here.

Credentials are stored locally and only sent to Twitch/IGDB.

Cache Management

Platform availability data is cached for 7 days to minimize requests.

Privacy

This extension:
  • Only communicates with Steam, Wikidata, and optionally Twitch/IGDB
  • Does not log or transmit your wishlist contents
  • Stores all data locally in your browser
  • Contains no telemetry or analytics
